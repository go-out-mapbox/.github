# This is a web form for submit your location.

このウェブフォームから、あなたがいた場所をコレクションに追加できます。
1. マーカー[^1] をドラッグ移動して、あなたがいた場所の座標・住所[^2] を取得
2. あなたがそこにいた日時と、その場所についてのコメントを入力[^3]
3. I Was Here を クリックして、コレクションに位置情報を追加

[^1]:位置情報APIの使用を許可・コントローラーから場所を検索すると、地図上にマーカーが追加されます。
[^2]:マーカーをドラッグ移動すると、あなたがいた場所を示す位置情報が更新されます。
[^3]:位置情報をクリックすると、日時・テキストを入力する投稿フォームが表示されます。

---

Mapbox Geocoding API は、座標を住所に(フォワードジオコーディング)、または住所を座標に(リバースジオコーディング) 変換するプログラムで、コントロールへの入力から住所・場所を検索し、位置情報・地理座標を取得します。
